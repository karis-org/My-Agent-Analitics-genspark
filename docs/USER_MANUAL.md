## このアプリについて

### 何ができるアプリ？

**My Agent Analytics**は、不動産投資の分析を簡単にするアプリです。

**できること:**
- 物件情報を入力するだけで、投資判断に必要な7つの指標を自動計算
- 国土交通省の実際の取引データを使って、市場の相場を調べられる
- 投資のリスクを自動で評価してくれる
- 分析結果をレポート形式で保存できる

**例えば...**
「この物件、買っても大丈夫かな？」
→ 物件情報を入力すると、利回りやリスクを自動で計算してくれます！

ブラウザで以下のURLを開いてください:

```
https://e594a8b5.my-agent-analytics.pages.dev/auth/login
```

**注意**: 上記は最新の本番環境URLです（2025年11月8日時点）。

**スマホの場合:**
1. 上記のURLをタップ
2. ブラウザが開きます

**パソコンの場合:**
1. ChromeやEdgeなどのブラウザを開く
2. 上記のURLをコピーして貼り付け
3. Enterキーを押す

### ステップ2-A: Googleアカウントでログイン（一般ユーザー）

画面に表示される「**Googleでログイン**」ボタンをクリック。

#### ⚠️ 重要: Google OAuth設定が必要です

現在、Googleログインを使用するには管理者がAPIキーを設定する必要があります。
→ `GOOGLE_OAUTH_SETUP.md` ファイルを参照して設定してください。

---

### ステップ2-B: 管理者としてログイン（管理者専用）

#### 管理者ログイン情報

画面下部のフォームに以下を入力:

```
メールアドレス: admin@myagent.local
パスワード: Admin@2025
```

**入力手順:**
1. 「メールアドレス」の欄をクリック
2. `admin@myagent.local` と入力
3. 「パスワード」の欄をクリック
4. `Admin@2025` と入力
5. 「管理者ログイン」ボタンをクリック

**注意事項:**
- 半角英数字で入力してください
- スペース（空白）を入れないでください
- 大文字・小文字を正確に入力してください

### ステップ3: ダッシュボード画面へ

ログインに成功すると、**ダッシュボード**画面が表示されます。

## 基本的な使い方

### 1. 物件を登録する

#### 1-1. 「新規物件」ボタンをクリック

ダッシュボード画面の右上にある「**+ 新規物件**」ボタンをクリック。

#### 1-2. 物件情報を入力

以下の情報を入力します:

**基本情報:**
- **物件名**: わかりやすい名前（例: 渋谷マンション）
- **物件価格**: 購入価格（例: 50000000 → 5000万円）
- **住所**: 物件の場所

**収益情報:**
- **年間家賃収入**: 1年間の家賃総額（例: 5000000 → 500万円）
- **空室率**: 何％空室があるか（例: 5 → 5%）

**費用情報:**
- **管理費**: 1年間の管理費
- **修繕費**: 1年間の修繕費
- **固定資産税**: 1年間の税金

**ローン情報:**
- **ローン額**: 借りる金額
- **金利**: 年利（例: 2.5 → 2.5%）
- **返済期間**: 何年で返すか（例: 30 → 30年）

#### 1-3. 「分析する」ボタンをクリック

入力が終わったら、画面下部の「**分析する**」ボタンをクリック。

---

### 2. 分析結果を見る

ボタンを押すと、すぐに分析結果が表示されます。

#### 表示される指標:

1. **NOI（純営業利益）**
   - 家賃収入から費用を引いた利益
   - 高いほど良い

2. **表面利回り**
   - 物件価格に対する家賃収入の割合
   - 目安: 都市部5-8%、地方8-12%

3. **実質利回り**
   - 費用を考慮した本当の利回り
   - 表面利回りより2-3%低いのが普通

4. **DSCR（債務償還カバー率）**
   - ローン返済の安全性
   - 目安: 1.25以上が理想

5. **LTV（ローン比率）**
   - 物件価格に対するローンの割合
   - 目安: 70%以下が安全

6. **CCR（自己資金利益率）**
   - 自分のお金に対する利益の割合
   - 目安: 8%以上が優良

7. **BER（損益分岐点）**
   - 赤字にならないギリギリのライン
   - 目安: 85%以下が理想

#### リスク評価:

画面に以下のいずれかが表示されます:
- 🟢 **低リスク**: 安全な投資
- 🟡 **中リスク**: 注意が必要
- 🔴 **高リスク**: 慎重に検討すべき

---

### 3. 市場相場を調べる

#### 3-1. 「市場分析」メニューをクリック

画面左側のメニューから「**📊 市場分析**」をクリック。

#### 3-2. 地域を選択

調べたい地域を選択:
1. 都道府県を選ぶ（例: 東京都）
2. 市区町村を選ぶ（例: 中央区）
3. 「検索」ボタンをクリック

#### 3-3. 結果を確認

以下の情報が表示されます:
- **平均取引価格**: この地域の平均価格
- **平均坪単価**: 1坪あたりの価格
- **取引件数**: 最近の取引の数
- **価格トレンド**: 価格が上がっているか、下がっているか

## よくある質問

### Q1: ログインできません

**A: 以下を確認してください:**

#### Googleログインの場合:
1. インターネットに接続されていますか？
2. Google OAuthの設定は完了していますか？
   → `GOOGLE_OAUTH_SETUP.md` を参照

#### 管理者ログインの場合:
1. メールアドレスは `admin@myagent.local` ですか？
2. パスワードは `Admin@2025` ですか？
3. 半角英数字で入力していますか？
4. スペースが入っていませんか？

**それでもダメな場合:**
- ブラウザを再起動してみてください
- キャッシュをクリアしてみてください（Ctrl+Shift+Delete）

---

### Q2: 分析結果がおかしい

**A: 入力値を確認してください:**

#### よくある間違い:
- 金額に「円」や「万円」を付けている
  - ❌ 5000万円
  - ✅ 50000000

- パーセントに「%」を付けている
  - ❌ 5%
  - ✅ 5

- 全角数字で入力している
  - ❌ ５０００
  - ✅ 5000

**正しい入力例:**
```
物件価格: 50000000（5000万円の場合）
年間家賃: 5000000（500万円の場合）
空室率: 5（5%の場合）
```

---

### Q3: 市場分析でデータが表示されない

**A: 以下を確認してください:**

1. **REINFOLIB APIキーは設定されていますか？**
   - システム情報ページで確認できます
   - 設定されていない場合、管理者に連絡してください

2. **選択した地域にデータがありますか？**
   - 地域によってはデータが少ない場合があります
   - 別の地域で試してみてください

3. **年度は正しいですか？**
   - 2005年以降のデータが利用可能です
   - 最新の年度を選択してください

---

### Q4: レポートをダウンロードしたい

**A: 現在準備中です**

この機能は次のバージョンで追加予定です。
現在は、画面のスクリーンショットを撮って保存してください。

**スクリーンショットの撮り方:**

#### Windowsの場合:
1. 保存したい画面を表示
2. `Win + Shift + S` キーを同時に押す
3. 保存したい範囲をドラッグして選択
4. 画像が自動的にクリップボードにコピーされます

#### Macの場合:
1. 保存したい画面を表示
2. `Cmd + Shift + 4` キーを同時に押す
3. 保存したい範囲をドラッグして選択
4. デスクトップに画像ファイルが保存されます

#### スマホの場合:
- iPhoneサイドボタンと音量上ボタンを同時に押す
- Android: 電源ボタンと音量下ボタンを同時に押す

---

### Q5: パスワードを変更したい

**A: データベースを直接編集する必要があります**

これは高度な操作なので、管理者に依頼することをおすすめします。

**自分で変更する場合:**

1. 新しいパスワードのハッシュを生成
```bash
node -e "const crypto = require('crypto'); console.log(crypto.createHash('sha256').update('新しいパスワード').digest('hex'));"
```

2. データベースを更新
```bash
npx wrangler d1 execute webapp-production --local \
  --command="UPDATE users SET password_hash='ハッシュ値' WHERE email='admin@myagent.local'"
```

## 用語解説

### 不動産投資の用語

- **NOI**: 家賃収入から費用を引いた純利益
- **利回り**: 投資額に対する収益の割合
- **DSCR**: ローン返済の安全性を示す指標
- **LTV**: ローンの割合（高いほどリスク大）
- **CCR**: 自己資金の効率性を示す指標

### システムの用語

- **API**: アプリ同士が情報をやり取りする仕組み
- **ダッシュボード**: トップ画面（全体が見える画面）
- **マイグレーション**: データベースの構造を変更すること
- **セッション**: ログイン状態を保つ仕組み
