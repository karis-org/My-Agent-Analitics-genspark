## はじめに

### My Agent Analyticsとは？

**My Agent Analytics**は、AIと政府統計データを活用した不動産投資分析プラットフォームです。

### 主な特徴

- **🤖 AI分析**: OpenAI GPT-4oによる高度な市場分析
- **📊 自動計算**: NOI、利回り、DSCR、LTVなど7つの投資指標を自動算出
- **📈 実データ分析**: 国土交通省の実取引価格データで市場動向を分析
- **🔍 事故物件調査**: AI搭載の心理的瑕疵調査システム
- **📄 PDFレポート**: プロフェッショナルなレポート生成
- **🏢 賃貸相場分析**: イタンジBB APIによる賃貸市場分析
- **📱 PWA対応**: スマートフォンにインストール可能

### 対応デバイス

- 💻 **デスクトップ**: Windows、Mac、Linux
- 📱 **モバイル**: iOS、Android
- 🌐 **ブラウザ**: Chrome、Edge、Safari、Firefox

## 物件管理

### 物件の新規登録

#### 手順1: 登録ページへ移動
1. ダッシュボードから「**物件を登録**」をクリック
2. または `/properties/new` にアクセス

#### 手順2: 基本情報の入力

**必須項目**:
- **物件名**: 識別しやすい名前（例: 恵比寿マンション301号室）
- **住所**: 都道府県・市区町村・番地
- **物件価格**: 購入価格（円）
- **構造**: RC、鉄骨、木造など
- **築年数**: 建築からの経過年数

**任意項目**:
- 専有面積
- 土地面積
- 間取り
- 最寄り駅
- 駅からの距離

#### 手順3: 収益情報の入力（投資物件の場合）

**収入**:
- **満室想定賃料**: 年間の想定家賃収入
- **実効賃料**: 空室を考慮した実際の収入

**支出**:
- **管理費**: 月額管理費
- **修繕積立金**: 月額積立金
- **固定資産税**: 年額
- **その他経費**: その他の年間経費

#### 手順4: ローン情報の入力（融資利用の場合）

- **借入金額**: ローン総額
- **金利**: 年利（%）
- **返済期間**: 年数
- **頭金**: 自己資金額

#### 手順5: 登録完了

「**登録**」ボタンをクリックすると、物件が保存され、自動的に統合レポートページへ遷移します。

## 投資指標計算

### 計算できる指標

#### 1. NOI（純営業利益）
**計算式**: `実効賃料 - 運営経費`

**意味**: 物件が生み出す純粋な収益

#### 2. 表面利回り
**計算式**: `(満室想定賃料 ÷ 物件価格) × 100`

**意味**: 単純な投資利回り

#### 3. 実質利回り
**計算式**: `(NOI ÷ 物件価格) × 100`

**意味**: 経費を考慮した実際の利回り

#### 4. DSCR（債務償還カバー率）
**計算式**: `NOI ÷ 年間返済額`

**意味**: ローン返済能力の指標
- 1.0以上: 返済可能
- 1.3以上: 理想的
- 1.0未満: 危険

#### 5. LTV（ローン対物件価値比率）
**計算式**: `(借入金額 ÷ 物件価格) × 100`

**意味**: レバレッジの度合い
- 70%以下: 安全
- 70-80%: やや高い
- 80%以上: 高リスク

#### 6. CCR（キャッシュ・オン・キャッシュ・リターン）
**計算式**: `(年間キャッシュフロー ÷ 自己資金) × 100`

**意味**: 自己資金に対するリターン

#### 7. BER（損益分岐点比率）
**計算式**: `(運営経費 ÷ 満室想定賃料) × 100`

**意味**: 空室リスクの指標
- 70%以下: 安全
- 70-80%: やや高い
- 80%以上: 高リスク

### ℹ️ ツールチップ機能

各指標名の横にある「❓」アイコンをクリックすると、詳細な説明が表示されます。

**表示例**:
```
LTV (Loan to Value / 融資比率)
物件価格に対する借入金額の割合です。
一般的に70%以下が安全とされています。
```

### ファクトチェック（レポート検証）

AIによるレポート内容の検証システムです。

#### アクセス方法
- ダッシュボードから「**ファクトチェック**」をクリック
- または `/fact-check` にアクセス

#### 検証観点（5つ）

1. **数値の妥当性**: 価格、面積、利回りが現実的か
2. **論理の一貫性**: 結論と根拠が矛盾していないか
3. **情報の完全性**: 必要な情報が欠けていないか
4. **リスク評価の適切性**: 過大/過小評価されていないか
5. **表現の正確性**: 誤解を招く表現がないか

#### 検証結果

**信頼度スコア**（0-100点）:
- **90点以上**: 非常に高い（十分に検証済み）
- **75-89点**: 高い（概ね検証済み）
- **60-74点**: 中程度（一部要確認）
- **45-59点**: 低い（複数の問題あり）
- **44点以下**: 非常に低い（大幅な見直し必要）

**警告システム**:
- 重要度レベル: info/warning/error
- 問題点の指摘
- 改善提案

**検証済み主張リスト**:
- 各主張の検証結果
- 信頼度
- 情報源
- 注記

**推奨事項**: 改善アクションの提示

1. PDFを画像形式（JPG、PNG）に変換
2. PDFをスクリーンショットで撮影
3. 手動で入力

### Q3: 市場分析で「データが見つかりません」と表示されます

**A**: 以下を確認してください:
1. 市区町村コードが正しいか
2. 対象年が2005年以降か
3. 東京23区または主要都市を選択しているか

### Q4: 統合レポートで「Property not found」エラーが出ます

**A**: 以下を確認してください:
1. 物件が削除されていないか
2. URLのIDが正しいか
3. アクセス権限があるか
4. ブラウザを再読み込み（Ctrl+F5）

### Q5: ロゴの背景が黒く表示されます

**A**: v6.7.4でSVG形式に変更しました。キャッシュクリアをお試しください:
- Chrome: Ctrl + Shift + R
- Safari: Cmd + Shift + R

### Q6: 事故物件調査でデモモードと表示されます

**A**: OpenAI APIキーが設定されていません。実際の調査を行うには、管理者にAPIキー設定を依頼してください。

### Q7: データをエクスポートしたい

**A**: 各機能ページに「エクスポート」ボタンがあります。CSV形式またはExcel形式を選択してダウンロードできます。

### Q8: 物件を編集できません

**A**: v6.7.3以降、物件編集機能が実装されました。物件詳細ページの「編集」ボタンをクリックしてください。

## トラブルシューティング

### 🔧 一般的な問題

#### 問題: ページが表示されない

**解決方法**:
1. ブラウザを最新バージョンに更新
2. キャッシュをクリア（Ctrl + Shift + Delete）
3. 別のブラウザで試す
4. ネットワーク接続を確認

#### 問題: 変更が反映されない

**解決方法**:
1. スーパーリロード:
   - Windows: Ctrl + Shift + R
   - Mac: Cmd + Shift + R
2. ブラウザキャッシュをクリア
3. 別のブラウザで確認

#### 問題: エラーメッセージが表示される

**解決方法**:
1. エラーメッセージをスクリーンショット
2. ブラウザのコンソールを確認（F12キー）
3. 管理者に問い合わせ

### 🚨 エラーコード一覧

| エラーコード | 意味 | 対処法 |
|-------------|------|--------|
| **API_KEY_NOT_CONFIGURED** | APIキー未設定 | 管理者に設定を依頼 |
| **INVALID_API_KEY** | 無効なAPIキー | 管理者に確認を依頼 |
| **RATE_LIMIT_EXCEEDED** | レート制限超過 | しばらく待ってから再試行 |
| **INVALID_IMAGE_FORMAT** | 画像形式エラー | JPG/PNG形式を使用 |
| **PDF_NOT_SUPPORTED** | PDF非対応 | 画像形式に変換 |
| **PROPERTY_NOT_FOUND** | 物件が見つからない | 物件ID、権限を確認 |
| **UNAUTHORIZED** | 認証エラー | ログインし直す |
